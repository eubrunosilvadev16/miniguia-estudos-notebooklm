# Segundo Cérebro com IA: Como se destacar no LinkedIn como Analista de Dados Júnior

## Contexto
Com o aumento da competitividade na área de dados, apenas possuir habilidades técnicas não é suficiente. Profissionais que se destacam no LinkedIn conseguem atrair oportunidades mesmo com pouca experiência.

Este projeto utiliza o :contentReference[oaicite:1]{index=1} para construir um "segundo cérebro" focado em posicionamento estratégico no LinkedIn, com base em fontes reais, análise de vagas e engenharia de prompts.

## Objetivos
- Identificar padrões em vagas de Analista de Dados Júnior
- Entender quais habilidades são mais valorizadas
- Criar uma estratégia de conteúdo para LinkedIn
- Desenvolver prompts reutilizáveis para geração de conteúdo
- Consolidar aprendizados em um sistema escalável de estudo
